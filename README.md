## Instalação

Instruções para instalar o projeto localmente.

```bash
# Clone o repositório
git clone https://github.com/euliveiras/fipe-price-query.git

# Entre no diretório do projeto
cd fipe-price-query

# Instale as dependências
npm install
```

## Uso

```bash
# Execute o projeto
npm run dev
```

```bash
# Para rodar os testes
npm run test
```
